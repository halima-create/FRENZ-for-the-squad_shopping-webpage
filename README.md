# FRENZ for the squad

Streetwear culture, desi prices. Grab the drip before it's gone.


A modern, fully responsive e-commerce website built using pure HTML, CSS, and JavaScript.  
Designed to deliver a smooth, fast, and stylish shopping experience with clean UI and interactive features.

✨ Built with simplicity, performance, and user experience in mind.



## 🚀 Features

- 🏠 Clean and attractive home page  
- 🛍️ Product listing with category filtering  
- ❤️ Wishlist functionality  
- 🛒 Cart with real-time updates  
- 🔍 Instant search feature  
- 📱 Fully responsive design
  

## 💻 Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  



## 💾 Storage

- Uses **localStorage** to store cart, wishlist, and orders  
- Data persists even after page refresh  



## ⚙️ How to Run

1. Download or clone the project  
2. Open `index.html` in your browser  



## 🎯 About the Project

This project demonstrates how a complete e-commerce frontend can be built using only HTML, CSS, and JavaScript without any frameworks or backend.
